# 📖 Подробная инструкция по админ-панели Telegram Shift Bot

## 🚀 Быстрый старт

### Как открыть админ-панель?

1. Откройте бота в Telegram
2. Отправьте команду `/admin`
3. Появится меню с кнопками — это и есть админ-панель!

---

## 📋 Основные разделы

Админ-панель состоит из 5 основных разделов:

1. **📋 Управление группами** — создание и настройка групп
2. **⚙️ Настройки** — расписание и слоты
3. **📊 Опросы** — управление опросами и просмотр результатов
4. **📢 Рассылка** — отправка сообщений во все группы
5. **📈 Мониторинг** — статистика, логи, верификация пользователей

---

## 📋 Раздел 1: Управление группами

### Что можно делать:
- ➕ Создать новую группу
- 📋 Посмотреть список всех групп
- 📌 Настроить темы для форум-групп
- ✏️ Переименовать группу
- 🗑️ Удалить группу

---

### ➕ Создание новой группы

**Когда использовать:** Когда нужно добавить новую группу для опросов (например, ЗИЗ-15)

**Пошаговая инструкция:**

1. Нажмите "📋 Управление группами"
2. Нажмите "➕ Создать группу"
3. Бот попросит ввести название группы
   - Пример: `ЗИЗ-1` или `НОЧЬ-лево`
4. Затем нужно ввести Chat ID группы
   - **Как получить Chat ID:**
     - Добавьте бота в группу и отправьте любое сообщение
     - Chat ID будет показан в логах
     - Или используйте бота `@userinfobot`
   - Chat ID обычно начинается с `-100` (например: `-1001234567890`)
5. Если группа форумная, бот спросит про Topic ID
   - Можно указать сразу или настроить позже через меню "📌 Установить тему"
6. Готово! Группа создана ✅

**Важно:** После создания группы нужно настроить слоты (см. раздел "⚙️ Настройки")

---

### 📋 Просмотр списка групп

**Когда использовать:** Чтобы увидеть все зарегистрированные группы и их статус

**Как использовать:**

1. Нажмите "📋 Управление группами"
2. Нажмите "📋 Список групп"
3. Вы увидите список всех групп с информацией:
   - ✅/❌ — активна ли группа
   - ☀️/🌙 — дневная или ночная группа
   - ID группы
   - Chat ID
   - Количество настроенных слотов
   - Время закрытия опросов

---

### 📌 Настройка тем для форум-групп

**Когда использовать:** Когда нужно настроить, в какой теме создавать опросы

**Пошаговая инструкция:**

1. Нажмите "📋 Управление группами"
2. Нажмите "📌 Установить тему"
3. Выберите тип темы:
   - **📊 Отметки на слот** — основная тема для опросов
   - **🚪 Приход/уход** — тема для скриншотов
   - **💬 Общий чат** — тема для уведомлений
   - **📢 Важная информация** — тема для важных сообщений
4. Выберите группу из списка
5. Бот попросит ввести Topic ID
   - **Как получить Topic ID:**
     - Откройте нужную тему в форум-группе
     - Скопируйте ссылку на тему (должна быть вида `https://t.me/c/1234567890/123`)
     - Последнее число в ссылке — это Topic ID
     - Или используйте команду `/get_topic_id` прямо в теме — бот покажет ID автоматически
6. Готово! Тема настроена ✅

---

### ✏️ Переименование группы

**Когда использовать:** Когда нужно изменить название группы (например, "ЗИЗ-1 (тест)" → "ЗИЗ-1")

**Пошаговая инструкция:**

1. Нажмите "📋 Управление группами"
2. Нажмите "✏️ Переименовать группу"
3. Выберите группу из списка
4. Введите новое название группы
5. Готово! Группа переименована ✅

---

### 🗑️ Удаление группы

**Когда использовать:** Когда группа больше не нужна

**⚠️ Внимание:** Удаление группы необратимо! Все данные (опросы, голоса) останутся в базе, но группа будет удалена.

**Пошаговая инструкция:**

1. Нажмите "📋 Управление группами"
2. Нажмите "🗑️ Удалить группу"
3. Выберите группу из списка
4. Подтвердите удаление
5. Группа удалена ✅

---

## ⚙️ Раздел 2: Настройки

### Что можно делать:
- ⏰ Настроить расписание (время создания и закрытия опросов)
- ⚙️ Настроить слоты для групп (время работы и количество курьеров)

---

### ⏰ Настройка расписания

**Когда использовать:** Когда нужно изменить время автоматического создания или закрытия опросов

**Пошаговая инструкция:**

1. Нажмите "⚙️ Настройки"
2. Нажмите "⏰ Настроить расписание"
3. Выберите тип расписания:
   - **📅 Создание опросов** — когда создавать новые опросы
   - **🔒 Закрытие опросов** — когда закрывать опросы
4. Выберите группу (или "Для всех групп")
5. Введите время в формате `ЧЧ:ММ` (например: `08:00`)
6. Готово! Расписание настроено ✅

**Примеры:**
- Создание опросов: `08:00` (каждый день в 8 утра)
- Закрытие опросов: `20:00` (каждый день в 8 вечера)

---

### ⚙️ Настройка слотов

**Когда использовать:** Когда нужно настроить рабочие смены для группы (например, утро 8:00-14:00, вечер 14:00-20:00)

**Пошаговая инструкция:**

1. Нажмите "⚙️ Настройки"
2. Нажмите "⚙️ Настроить слоты"
3. Выберите группу из списка
4. Выберите действие:
   - **➕ Добавить слот** — добавить новый временной слот
   - **✏️ Изменить слот** — изменить существующий слот
   - **🗑️ Удалить слот** — удалить слот
5. Для добавления/изменения:
   - Введите время начала (формат: `ЧЧ:ММ`)
   - Введите время окончания (формат: `ЧЧ:ММ`)
   - Введите количество курьеров (опционально)
6. Готово! Слот настроен ✅

**Примеры слотов:**
- Утро: `08:00` - `14:00`, 5 курьеров
- Вечер: `14:00` - `20:00`, 3 курьера
- Ночь: `20:00` - `08:00`, 2 курьера

## 📊 Раздел 3: Опросы

### Что можно делать:
- 📊 Просмотреть результаты опросов
- 🔄 Создать опрос вручную
- 🔒 Закрыть опрос досрочно
- 📈 Статистика по опросам

---

### 📊 Просмотр результатов опросов

**Когда использовать:** Чтобы увидеть, кто отметился на слоты

**Пошаговая инструкция:**

1. Нажмите "📊 Опросы"
2. Нажмите "📊 Результаты опросов"
3. Выберите группу
4. Выберите дату (или "Сегодня")
5. Вы увидите:
   - Список всех опросов за выбранную дату
   - Количество голосов по каждому слоту
   - Список курьеров, отметившихся на каждый слот

---

### 🔄 Создание опроса вручную

**Когда использовать:** Когда нужно создать опрос вне расписания

**Пошаговая инструкция:**

1. Нажмите "📊 Опросы"
2. Нажмите "🔄 Создать опрос"
3. Выберите группу
4. Выберите дату для опроса
5. Бот создаст опрос со всеми настроенными слотами
6. Готово! Опрос создан ✅

---

### 🔒 Закрытие опроса досрочно

**Когда использовать:** Когда нужно закрыть опрос раньше запланированного времени

**Пошаговая инструкция:**

1. Нажмите "📊 Опросы"
2. Нажмите "🔒 Закрыть опрос"
3. Выберите группу
4. Выберите активный опрос
5. Подтвердите закрытие
6. Опрос закрыт ✅

---

## 📢 Раздел 4: Рассылка

### Что можно делать:
- 📢 Отправить сообщение во все группы
- 📋 Отправить сообщение в конкретную группу
- 📅 Запланировать рассылку

---

### 📢 Отправка сообщения во все группы

**Когда использовать:** Когда нужно уведомить всех курьеров о важной информации

**Пошаговая инструкция:**

1. Нажмите "📢 Рассылка"
2. Нажмите "📢 Отправить во все группы"
3. Введите текст сообщения
4. Выберите тип сообщения:
   - **Обычное сообщение** — просто текст
   - **С форматированием** — с поддержкой Markdown/HTML
   - **С кнопками** — с inline-кнопками (опционально)
5. Подтвердите отправку
6. Сообщение отправлено во все активные группы ✅

---

### 📋 Отправка в конкретную группу

**Когда использовать:** Когда сообщение предназначено только для одной группы

**Пошаговая инструкция:**

1. Нажмите "📢 Рассылка"
2. Нажмите "📋 Отправить в группу"
3. Выберите группу из списка
4. Введите текст сообщения
5. Подтвердите отправку
6. Сообщение отправлено ✅

---

### Что можно делать:

- 📝 Просмотреть логи
- ✅ Верифицировать пользователей
- 🔍 Поиск по логам

---

### 📊 Просмотр статистики

**Когда использовать:** Чтобы увидеть общую статистику работы бота

**Пошаговая инструкция:**

1. Нажмите "📈 Мониторинг"
2. Нажмите "📊 Статистика"
3. Выберите период:
   - Сегодня
   - Неделя
   - Месяц
   - Все время
4. Вы увидите:
   - Количество созданных опросов
   - Количество активных групп
   - Количество пользователей
   - Средняя активность

---

### 📝 Просмотр логов

**Когда использовать:** Для отладки и поиска проблем

**Пошаговая инструкция:**

1. Нажмите "📈 Мониторинг"
2. Нажмите "📝 Логи"
3. Выберите уровень логов:
   - Все
   - Ошибки
   - Предупреждения
   - Информация
4. Выберите период
5. Логи будут показаны в хронологическом порядке

---

### ✅ Верификация пользователей

**Когда использовать:** Когда нужно дать пользователю доступ к админ-панели

**Пошаговая инструкция:**

1. Нажмите "📈 Мониторинг"
2. Нажмите "✅ Верификация"
3. Выберите действие:
   - **➕ Добавить админа** — дать права администратора
   - **📋 Список админов** — посмотреть всех администраторов
   - **🗑️ Удалить админа** — убрать права администратора
4. Для добавления:
   - Введите Telegram ID пользователя
   - Или перешлите сообщение от пользователя
5. Готово! Пользователь верифицирован ✅

## 🔐 Безопасность

### Права доступа

- **Администратор** — полный доступ ко всем функциям
- **Модератор** — ограниченный доступ (только просмотр)
- **Пользователь** — только участие в опросах

### Рекомендации

- Не делитесь доступом к админ-панели с посторонними
- Регулярно проверяйте список администраторов
- Используйте сильные пароли для базы данных
- Делайте резервные копии данных

## ❓ Часто задаваемые вопросы (FAQ)

### Как получить Chat ID группы?

1. Добавьте бота `@userinfobot` в группу
2. Отправьте команду `/start`
3. Бот покажет Chat ID группы

Или:
1. Добавьте вашего бота в группу
2. Отправьте любое сообщение
3. Chat ID будет в логах бота

### Как получить Topic ID?

1. Откройте нужную тему в форум-группе
2. Скопируйте ссылку на тему
3. Последнее число в ссылке — это Topic ID

Или:
1. Используйте команду `/get_topic_id` прямо в теме
2. Бот покажет Topic ID автоматически

### Опросы не создаются автоматически

Проверьте:
1. ✅ Расписание настроено правильно
2. ✅ Группа активна
3. ✅ Бот добавлен в группу
4. ✅ Слоты настроены для группы
5. ✅ Бот имеет права на создание опросов в группе

### Как изменить время закрытия опросов?

1. Перейдите в "⚙️ Настройки"
2. Выберите "⏰ Настроить расписание"
3. Выберите "🔒 Закрытие опросов"
4. Введите новое время

2. Убедитесь, что все настройки корректны
3. Проверьте, что бот имеет необходимые права в группах
4. Обратитесь к разработчику с описанием проблемы

## 📝 Примечания

- Все изменения сохраняются автоматически
- Время указывается в формате 24 часа (ЧЧ:ММ)
- Часовой пояс: UTC+3 (Москва)
- Резервные копии создаются автоматически каждый день
