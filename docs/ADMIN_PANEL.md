# 🎛️ Админ-панель

## Разделы

1. `Управление группами` — создание, переименование, удаление групп ЗИЗ
2. `Сотрудники` — ручное ведение состава сотрудников по каждой группе, переименование, перенос между группами и контроль привязки к Telegram
3. `Настройки` — расписание и слоты
4. `Опросы` — ручное создание, пересоздание, просмотр, напоминания и закрытие
5. `Рассылка` — отправка сообщений по всем активным группам
6. `Мониторинг` — статистика, логи, верификация

## Базовый порядок настройки

1. Создайте группу ЗИЗ и сохраните `Chat ID`.
2. Добавьте сотрудников в раздел `Сотрудники`.
3. Для новой дневной группы стандартные слоты подставляются автоматически. При необходимости измените их в разделе `Настройки`.
4. Проверьте расписание:
- `09:00` — создание опроса
- `17:00` — напоминание неотметившимся
- `19:00` — итог и список неотметившихся для дневных групп
- `17:00` — итог для ночных групп

## Тест через админ-панель

В разделе `Опросы` доступны ручные действия для проверки:

- `Отправить тестовый опрос` — отправляет опрос в выбранную группу
- `Тест напоминания 17:00` — вручную отправляет в группу список неотметившихся с тегами
- `Результаты тестового опроса` — показывает последний опрос группы независимо от даты
- `Закрыть тестовый опрос` — закрывает последний активный опрос группы независимо от даты
- `Удалить тестовый опрос` — удаляет последний опрос группы из базы данных
- `Удалить все тестовые опросы` — удаляет все опросы выбранной группы из базы данных после подтверждения
- `Закрыть опрос` — вручную закрывает опрос выбранной группы и отправляет итог в чат как по сценарию
- `Закрыть все опросы` — вручную закрывает все активные опросы
- `Создать опросы всем` — запускает массовое создание опросов для всех активных групп
- `Пересоздать опросы` — удаляет активные опросы на нужную дату и создает их заново
- `Результаты опросов` — показывает текущие результаты по выбранной группе
- `Найти опросы на завтра` — быстро проверяет, в каких группах опрос уже есть

## Сценарий теста

1. Откройте `/admin` → `Управление группами` и создайте тестовую группу.
2. Откройте `/admin` → `Сотрудники` и добавьте 2-3 сотрудников в эту группу.
3. Откройте `/admin` → `Настройки` → `Настроить слоты` и задайте хотя бы один дневной вариант выхода.
4. Откройте `/admin` → `Опросы` → `Отправить тестовый опрос` и выберите группу.
5. Пусть один сотрудник проголосует, а второй нет.
6. Откройте `/admin` → `Опросы` → `Результаты опросов` и проверьте:
- кто уже отметился;
- кто выбрал `Куратор`;
- кто попал в список `Не отметились`.
7. Откройте `/admin` → `Опросы` → `Закрыть тестовый опрос` и закройте тестовый опрос вручную.
8. После закрытия откройте `Результаты тестового опроса` и убедитесь, что данные сохранились.
9. Если нужно начать заново, используйте `Удалить все тестовые опросы` для очистки истории группы.
10. Если в группе накопилось несколько активных тестовых опросов, при `Закрыть тестовый опрос` бот предложит выбрать конкретный опрос.

## Важно

- Тем внутри групп больше нет.
- У админа есть постоянная кнопка `Админ-панель` снизу, вход не только через `/admin`.
- Новая дневная группа создается сразу со стандартными слотами: `07:00-19:00`, `08:00-20:00`, `09:00-21:00`, `10:00-22:00`, `12:00-00:00`.
- Если в названии новой группы есть `ноч` или `night`, бот автоматически считает ее ночной группой.
- В разделе `Сотрудники` можно:
  - добавить сотрудника вручную;
  - переименовать сотрудника и записать корректные `Имя Фамилия` в базу;
  - перенести сотрудника из одной группы в другую;
  - удалить сотрудника из группы без потери записи — он становится неактивным.
- В каждом дневном опросе есть варианты `Куратор` и `Выходной` по умолчанию.
- Для дневной группы заголовок опроса: `Выходы на завтра`.
- Для ночной группы заголовок опроса: `Выходы на сегодня`.
- Для ночной группы опрос состоит из вариантов `Выхожу`, `Не выхожу`, `Куратор`, `Выходной`.
- При создании опрос автоматически закрепляется в группе с уведомлением участников.
- Список неотметившихся считается только по сотрудникам, которые заведены в реестр группы.
- При первом голосовании Telegram-пользователь привязывается к записи сотрудника.
- Если сотрудник уже был привязан к Telegram и проголосовал в другой группе, бот автоматически перенесет его в новую группу.
- В 17:00 бот отправляет в группу список неотметившихся с тегами.
