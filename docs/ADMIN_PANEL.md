# 🎛️ Админ-панель Telegram Shift Bot

Понятное руководство для администраторов групп ЗИЗ.  
Ниже описан весь рабочий функционал админ-панели простым языком.

---

## 👑 Как зайти в админку

Есть два способа входа:

- команда `/admin`
- постоянная кнопка `👑 Админ-панель` внизу Telegram

Важно:

- меню админки видят только пользователи из `ADMIN_IDS`
- если кнопка уже показана, можно работать без команд

---

## 🧭 Структура админ-панели

Главное меню:

- `📋 Управление группами`
- `⚙️ Настройки`
- `📊 Опросы`
- `📢 Рассылка`
- `📈 Мониторинг`

Отдельный раздел `👥 Сотрудники` находится внутри `📋 Управление группами`.

---

## ⚙️ Логика работы бота

### ☀️ Дневные группы

- в `09:00` бот создает опрос на **завтра**
- в `17:00` бот отправляет напоминание тем, кто не отметился
- в `19:00` бот закрывает опрос
- после закрытия бот отправляет:
  - результаты по выходам
  - результаты по `Куратор`
  - результаты по `Выходной`
  - список тех, кто не отметился

### 🌙 Ночные группы

- в `09:00` бот создает опрос на **сегодня**
- в `12:00` бот отправляет напоминание тем, кто не отметился
- в `17:00` бот закрывает опрос
- после закрытия бот отправляет:
  - результаты по `Выхожу`
  - результаты по `Не выхожу`
  - результаты по `Куратор`
  - результаты по `Выходной`
  - список тех, кто не отметился

### 👮 Кто попадает в список `Не отметились`

В список попадают только сотрудники из реестра выбранной группы:

- кто заведён вручную в разделе `Сотрудники`
- кто ещё не проголосовал в активном опросе

Не попадают в упоминания:

- админы бота
- сотрудники, которых нет в реестре группы

## 📋 Управление группами

Раздел нужен для создания и обслуживания групп ЗИЗ.

### Доступные действия

- `➕ Создать группу`
- `📋 Список групп`
- `👥 Сотрудники`
- `✏️ Переименовать группу`
- `🗑️ Удалить группу`

### ➕ Создать группу

Что делает:

- создаёт новую группу в базе
- сохраняет `Chat ID`
- определяет, ночная группа или нет
- подставляет базовые настройки автоматически

Как работает:

1. Админ вводит название группы
2. Админ вводит `Chat ID`
3. Бот создаёт группу и сохраняет настройки

Важно:

- если в названии есть `ноч` или `night`, группа считается ночной
- темы внутри групп не используются
- группа работает как один общий чат

### 📋 Список групп

Показывает:

- все зарегистрированные группы
- формат отображения без служебного мусора
- пагинацию, если групп много

### ✏️ Переименовать группу

Позволяет:

- выбрать группу из списка
- задать новое название

### 🗑️ Удалить группу

Удаляет группу из рабочей конфигурации после подтверждения.

Использовать осторожно.

## 👥 Сотрудники

Раздел находится внутри `📋 Управление группами`.

### Доступные действия

- `➕ Добавить сотрудника`
- `📋 Список сотрудников`
- `✏️ Переименовать сотрудника`
- `🔄 Перенести в другую группу`
- `🔗 Статус привязки Telegram`
- `🗑️ Удалить сотрудника`

### ➕ Добавить сотрудника

Админ вручную вносит сотрудника в нужную группу.

Рекомендуемый формат:

- `Фамилия Имя`

Пример:

- `Иванов Иван`

### 📋 Список сотрудников

Показывает:

- активных сотрудников группы
- удалённых из группы сотрудников

### ✏️ Переименовать сотрудника

Нужно для приведения данных к правильному виду:

- записать корректные `Имя Фамилия`
- сразу обновить текстовые результаты опросов

### 🔄 Перенести в другую группу

Позволяет вручную перенести сотрудника между ЗИЗ.

### 🔗 Статус привязки Telegram

Показывает:

- кто уже привязан к Telegram
- кто ещё не привязан
- `telegram_user_id`
- `username`, если он есть

### 🗑️ Удалить сотрудника

Что происходит:

- сотрудник удаляется из активного состава группы
- запись остаётся в базе как неактивная

### 🤖 Автоматическая привязка сотрудника

Когда сотрудник голосует в опросе:

- бот пытается найти его в реестре группы
- если находит, привязывает к `telegram_user_id`
- если сотрудник уже был привязан и проголосовал в другой группе, бот автоматически переносит его в новую группу

Это позволяет вести актуальный состав без ручной перепривязки.

## ⚙️ Настройки

Раздел нужен для настройки расписания и вариантов ответов.

### Доступные действия

- `⏰ Настроить расписание`
- `⚙️ Настроить слоты`
- `📝 Доп. ответы в опросе`

### ⏰ Настроить расписание

Позволяет редактировать:

- время создания опросов
- время закрытия опросов
- часы напоминаний для дневных групп

Важно:

- ночное напоминание работает по отдельному сценарию в `12:00`
- закрытие ночных опросов идёт в `17:00`

### ⚙️ Настроить слоты

Используется только для дневных групп.

Что можно сделать:

- посмотреть текущие выходы
- создать новые выходы
- изменить время существующих выходов
- удалить лишние выходы

Как сейчас устроены дневные опросы:

- в опросе нет деления по количеству курьеров
- у варианта нет ограничения по числу голосов
- отображается только время выхода

Стандартные выходы для новой дневной группы:

- `07:00 - 19:00`
- `08:00 - 20:00`
- `09:00 - 21:00`
- `10:00 - 22:00`
- `12:00 - 00:00`

### 📝 Доп. ответы в опросе

Позволяет добавить нестандартные ответы в опрос для конкретной группы.

Примеры:

- `Обучение`
- `Стажировка`
- `Подмена`

Важно:

- доп. ответы попадают в опрос как отдельные варианты
- голоса по ним сохраняются в базу
- они отображаются в результатах и текстовых отчётах

## 📊 Опросы

Это главный рабочий раздел для контроля смен.

### Доступные действия

- `📝 Создать опросы всем`
- `📨 Отправить тестовый опрос`
- `🔔 Тест напоминания`
- `📊 Результаты тестового опроса`
- `🔒 Закрыть тестовый опрос`
- `🗑️ Удалить тестовый опрос`
- `🧹 Удалить все тестовые опросы`
- `🔄 Пересоздать опросы`
- `📊 Результаты опросов`
- `🔒 Закрыть опрос`
- `🔒 Закрыть все опросы`
- `🔎 Найти опросы на завтра`

### 📝 Создать опросы всем

Создаёт опросы для всех активных групп:

- для дневных — на завтра
- для ночных — на сегодня

После отправки:

- опрос сохраняется в базе
- сообщение с опросом закрепляется в группе
- закрепление идёт с уведомлением участников

### 📨 Отправить тестовый опрос

Нужно для ручной проверки одной конкретной группы.

Подходит для:

- теста перед запуском
- проверки новой группы
- проверки после изменений в настройках

### 🔔 Тест напоминания

Отправляет в группу сообщение со списком тех, кто не отметился:

- для дневной группы — сценарий `17:00`
- для ночной группы — сценарий `12:00`

Бот сам понимает тип группы.

### 📊 Результаты тестового опроса

Показывает последний опрос группы независимо от даты.

Удобно, когда:

- тестов было несколько
- нужно быстро посмотреть последний результат

### 🔒 Закрыть тестовый опрос

Закрывает последний активный тестовый опрос группы.

Если найдено несколько активных опросов:

- бот предложит выбрать конкретный

После закрытия бот:

- закрывает опрос в Telegram
- сохраняет итог в базу
- отправляет результат в чат
- отправляет список `Не отметились`

### 🗑️ Удалить тестовый опрос

Удаляет последний опрос группы из базы.

### 🧹 Удалить все тестовые опросы

Полностью очищает историю опросов выбранной группы в базе.

Нужно использовать аккуратно.

### 🔄 Пересоздать опросы

Сценарий:

- активные опросы удаляются
- новые опросы создаются заново

Нужно понимать:

- голоса в старых опросах будут потеряны

### 📊 Результаты опросов

Показывает рабочие результаты по нужной группе.

Для дневной группы выводятся:

- выходы по времени
- `Куратор`
- `Выходной`
- дополнительные ответы
- `Не отметились`

Для ночной группы выводятся:

- `Выхожу`
- `Не выхожу`
- `Куратор`
- `Выходной`
- дополнительные ответы
- `Не отметились`

### 🔒 Закрыть опрос

Ручное закрытие рабочего опроса.

Важно:

- для дневной группы закрывается опрос на завтра
- для ночной группы закрывается опрос на сегодня
- после закрытия результат уходит в чат так же, как при автоматическом сценарии

### 🔒 Закрыть все опросы

Закрывает все активные опросы сразу.

Использовать только если действительно нужно массовое ручное завершение.

### 🔎 Найти опросы на завтра

Показывает проверку наличия опросов:

- отдельно по дневным группам
- отдельно по ночным группам

Важно:

- для ночных групп проверяется опрос на **сегодня**
- для дневных групп проверяется опрос на **завтра**

## 📢 Рассылка

Раздел для отправки сообщений по всем активным группам.

### Что умеет

- отправка текста
- отправка фото с подписью
- HTML-разметка в тексте

### Как пользоваться

1. Открыть `📢 Рассылка`
2. Нажать `✉️ Новая рассылка`
3. Отправить текст или фото с подписью
4. Дождаться отчёта

После завершения бот показывает:

- сколько групп получили сообщение
- где возникли ошибки

## 📈 Мониторинг

Раздел для проверки состояния системы.

### Доступные действия

- `📊 Статистика`
- `🔍 Статус системы`
- `📜 Логи`
- `👤 Верификация`

### 📊 Статистика

Показывает:

- сколько всего групп
- сколько активных групп
- сколько дневных и ночных групп
- сколько активных опросов
- сколько опросов было за сегодня

### 🔍 Статус системы

Показывает:

- загрузку CPU
- использование RAM
- использование диска
- аптайм сервера

### 📜 Логи

Показывает последние строки рабочего лога бота.

Это основной раздел для быстрой диагностики ошибок.

### 👤 Верификация

Служебный раздел для пользователей:

- список неверифицированных
- список верифицированных
- массовая верификация
- переименование и удаление карточек пользователей

Для ежедневной работы ЗИЗ этот раздел обычно нужен редко.

## 🧪 Рекомендуемый порядок запуска новой группы

### Шаг 1. Создать группу

Открыть:

- `📋 Управление группами` → `➕ Создать группу`

Нужно ввести:

- название группы
- `Chat ID`

### Шаг 2. Проверить тип группы

Понять:

- группа дневная или ночная

Если это ночная группа, лучше сразу проверить сценарий `09:00 → 12:00 → 17:00`.

### Шаг 3. Заполнить сотрудников

Открыть:

- `📋 Управление группами` → `👥 Сотрудники`

Добавить:

- весь актуальный состав группы

### Шаг 4. Проверить выходы

Для дневной группы:

- открыть `⚙️ Настройки` → `⚙️ Настроить слоты`
- убедиться, что выходы актуальны

Для ночной группы:

- слоты не нужны

### Шаг 5. При необходимости добавить доп. ответы

Открыть:

- `⚙️ Настройки` → `📝 Доп. ответы в опросе`

### Шаг 6. Отправить тестовый опрос

Открыть:

- `📊 Опросы` → `📨 Отправить тестовый опрос`

### Шаг 7. Проверить голосование

Нужно убедиться, что:

- сотрудник видит опрос
- голос записывается
- сотрудник привязывается к Telegram
- результаты видны в админке

### Шаг 8. Проверить напоминание

Открыть:

- `📊 Опросы` → `🔔 Тест напоминания`

Проверить:

- в сообщении есть только неотметившиеся
- админы не тегаются

### Шаг 9. Проверить закрытие

Открыть:

- `📊 Опросы` → `🔒 Закрыть тестовый опрос`

Проверить:

- опрос закрылся
- результат отправлен в чат
- список `Не отметились` отправлен в чат

## ✅ Что важно помнить администраторам

- все группы работают без тем
- один чат группы используется и для опросов, и для результатов, и для рассылки
- сотрудники должны быть заведены в реестр, иначе бот не сможет корректно считать `Не отметились`
- переименование сотрудника сразу влияет на будущие текстовые результаты
- тестовые функции в админке нужны именно для безопасной проверки сценария
- ночные группы живут по дате `сегодня`, дневные — по дате `завтра`

## 🚨 Частые причины проблем

### Опрос есть в Telegram, но нет результата в админке

Проверьте:

- бот получил голосования
- опрос не был создан вручную вне логики бота
- у группы правильный `Chat ID`

### В списке `Не отметились` кто-то лишний

Проверьте:

- сотрудник точно есть в правильной группе
- сотрудник голосовал именно в этом опросе
- сотрудник не числится в другой группе

### Не получается создать группу

Проверьте:

- `Chat ID` введён числом
- группа с таким `Chat ID` ещё не зарегистрирована

### Не работает рассылка или опрос

Проверьте:

- бот добавлен в группу
- у бота есть права на отправку сообщений
- у бота есть права на закрепление сообщений
