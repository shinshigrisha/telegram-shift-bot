# 🎛️ Админ-панель

Полное руководство по использованию админ-панели Telegram Shift Bot.

---

## 🚀 Быстрый старт

### Открытие админ-панели

1. Откройте бота в Telegram
2. Отправьте команду `/admin`
3. Появится главное меню с разделами

**Важно:** Админ-панель доступна только пользователям, чьи ID указаны в `ADMIN_IDS` в `.env`

---

## 📋 Основные разделы

Админ-панель состоит из 6 основных разделов:

1. **📋 Управление группами** — создание и настройка групп
2. **⚙️ Настройки** — расписание и слоты
3. **📊 Опросы** — управление опросами и просмотр результатов
4. **🤖 AI куратор** — управление базой знаний и AI-куратором
5. **📢 Рассылка** — отправка сообщений во все группы
6. **📈 Мониторинг** — статистика, логи, верификация пользователей

---

## 📋 Раздел 1: Управление группами

### Возможности

- ➕ Создать новую группу
- 📋 Посмотреть список всех групп
- 📌 Настроить темы для форум-групп
- ✏️ Переименовать группу
- 🗑️ Удалить группу
- ✅ Активировать/деактивировать группу

### Создание новой группы

**Пошаговая инструкция:**

1. Нажмите "📋 Управление группами"
2. Нажмите "➕ Создать группу"
3. Введите название группы (например: `ЗИЗ-1` или `НОЧЬ-лево`)
4. Введите Chat ID группы
   - **Как получить Chat ID:**
     - Добавьте бота в группу и отправьте любое сообщение
     - Chat ID будет показан в логах
     - Или используйте бота `@userinfobot`
   - Chat ID обычно начинается с `-100` (например: `-1001234567890`)
5. Если группа форумная, введите Topic ID (можно пропустить и настроить позже)
6. Готово! Группа создана ✅

**Важно:** После создания группы нужно настроить слоты (см. раздел "⚙️ Настройки")

### Настройка тем для форум-групп

**Типы тем:**

- **📊 Отметки на слот** — основная тема для опросов
- **🚪 Приход/уход** — тема для скриншотов
- **💬 Общий чат** — тема для уведомлений
- **📢 Важная информация** — тема для важных сообщений

**Как получить Topic ID:**

1. Откройте нужную тему в форум-группе
2. Скопируйте ссылку на тему (должна быть вида `https://t.me/c/1234567890/123`)
3. Последнее число в ссылке — это Topic ID
4. Или используйте команду `/get_topic_id` прямо в теме — бот покажет ID автоматически

## ⚙️ Раздел 2: Настройки

### Расписание опросов

**Настройки:**

- **Время создания опросов** — по умолчанию 09:00
- **Время закрытия опросов** — по умолчанию 19:00
- **Часы напоминаний** — по умолчанию 10, 12, 14, 16, 18

**Изменение расписания:**

1. Нажмите "⚙️ Настройки"
2. Нажмите "⏰ Расписание опросов"
3. Выберите параметр для изменения
4. Введите новое значение

### Управление слотами

**Слот** — временной интервал для записи курьеров (например: 10:00-12:00, лимит 5 курьеров)

**Добавление слота:**

1. Нажмите "⚙️ Настройки"
2. Нажмите "📅 Слоты"
3. Выберите группу
4. Нажмите "➕ Добавить слот"
5. Введите время начала (например: `10:00`)
6. Введите время окончания (например: `12:00`)
7. Введите лимит курьеров (1-10)
8. Готово! Слот добавлен ✅

**Редактирование слота:**

1. Выберите группу
2. Выберите слот из списка
3. Нажмите "✏️ Редактировать"
4. Измените параметры
5. Сохраните изменения

**Удаление слота:**

1. Выберите группу
2. Выберите слот из списка
3. Нажмите "🗑️ Удалить"
4. Подтвердите удаление

## 📊 Раздел 3: Опросы

### Возможности

- ➕ Создать опрос вручную
- 🔄 Пересоздать опрос
- 📊 Просмотреть результаты опросов
- 🔒 Закрыть опрос (одиночное и массовое)
- 🔍 Поиск опросов на завтра

### Создание опроса вручную

**Когда использовать:** Когда нужно создать опрос не по расписанию

**Пошаговая инструкция:**

1. Нажмите "📊 Опросы"
2. Нажмите "➕ Создать опрос"
3. Выберите группу
4. Выберите дату (по умолчанию сегодня)
5. Бот создаст опрос автоматически на основе настроенных слотов

### Просмотр результатов опросов

**Информация в результатах:**

- Дата опроса
- Группа
- Слоты с количеством голосов
- Детализация по каждому слоту
- Имена курьеров, записавшихся на слот

**Как просмотреть:**

1. Нажмите "📊 Опросы"
2. Нажмите "📊 Результаты опросов"
3. Выберите группу
4. Выберите дату
5. Просмотрите результаты

### Закрытие опросов

**Одиночное закрытие:**

1. Выберите опрос из списка
2. Нажмите "🔒 Закрыть опрос"
3. Подтвердите закрытие

**Массовое закрытие:**

1. Нажмите "📊 Опросы"
2. Нажмите "🔒 Закрыть все опросы"
3. Подтвердите закрытие

## 🤖 Раздел 4: AI куратор

### Возможности

- ➕ Добавить FAQ в базу знаний
- 🔍 Поиск FAQ по запросу
- 📢 Создать информационное сообщение через AI
- ⚠️ Создать замечание курьеру через AI
- 🗑️ Очистить историю диалогов пользователя
- 📊 Статистика AI (количество FAQ, категории, теги)

### Добавление FAQ

**Пошаговая инструкция:**

1. Нажмите "🤖 AI куратор"
2. Нажмите "➕ Добавить FAQ"
3. Введите вопрос
4. Введите ответ
5. (Опционально) Укажите категорию и тег
6. Готово! FAQ добавлен в базу знаний ✅

### Поиск FAQ

1. Нажмите "🤖 AI куратор"
2. Нажмите "🔍 Поиск FAQ"
3. Введите поисковый запрос
4. Просмотрите найденные FAQ

### Создание информационного сообщения

**Когда использовать:** Когда нужно создать информационное сообщение для рассылки курьерам

**Пошаговая инструкция:**

1. Нажмите "🤖 AI куратор"
2. Нажмите "📢 Создать информационное сообщение"
3. Введите тему сообщения
4. AI сгенерирует текст на основе базы знаний
5. Проверьте и отправьте (или отредактируйте)

### Создание замечания курьеру

**Когда использовать:** Когда нужно создать замечание для конкретного курьера

**Пошаговая инструкция:**

1. Нажмите "🤖 AI куратор"
2. Нажмите "⚠️ Создать замечание"
3. Введите описание ситуации
4. Введите User ID курьера (или выберите из списка)
5. AI сгенерирует текст замечания
6. Проверьте и отправьте

### Очистка истории диалогов

**Когда использовать:** Когда нужно очистить историю диалогов пользователя с AI-куратором

1. Нажмите "🤖 AI куратор"
2. Нажмите "🗑️ Очистить историю"
3. Введите User ID пользователя
4. Подтвердите очистку

## 📢 Раздел 5: Рассылка

### Возможности

- Отправка текстовых сообщений во все группы
- Отправка фото с подписью
- Выбор темы для рассылки (для форум-групп)

### Типы тем для рассылки

- **📊 Отметки на слот** — основная тема для опросов
- **🚪 Приход/уход** — тема для скриншотов
- **💬 Общий чат** — тема для уведомлений
- **📢 Важная информация** — тема для важных сообщений

### Отправка сообщения

**Пошаговая инструкция:**

1. Нажмите "📢 Рассылка"
2. Выберите тип темы
3. Введите текст сообщения (или отправьте фото с подписью)
4. Подтвердите отправку
5. Просмотрите отчет о результатах (отправлено, пропущено, ошибки)

---

## 📈 Раздел 6: Мониторинг

### Возможности

- 📊 Статистика системы
- 💻 Системный статус (CPU, RAM, Disk)
- 📝 Просмотр логов
- 👥 Верификация пользователей

### Статистика системы

**Информация:**

- Общее количество групп (активных, дневных, ночных)
- Количество активных/закрытых опросов
- Статистика по голосам за сегодня

### Системный статус

**Информация:**

- CPU usage (%)
- RAM usage (%)
- Disk usage (%)
- System uptime

### Просмотр логов

**Логи:**

- Последние строки из `logs/bot.log`
- Логи explainability для AI-куратора (JSON)

### Верификация пользователей

**Возможности:**

- Просмотр списка неверифицированных пользователей
- Просмотр списка верифицированных пользователей
- Верификация пользователя (ввод имени и фамилии)
- Переименование верифицированного пользователя
- Удаление (unverify) пользователя
- Массовая верификация всех пользователей

**Верификация пользователя:**

1. Нажмите "📈 Мониторинг"
2. Нажмите "👥 Верификация пользователей"
3. Выберите "➕ Верифицировать пользователя"
4. Введите User ID
5. Введите имя и фамилию
6. Готово! Пользователь верифицирован ✅

## 🔒 Безопасность

### Проверка прав доступа

- Админ-панель доступна только пользователям из `ADMIN_IDS`
- Все операции проверяются через `AdminMiddleware`
- Callback queries защищены декоратором `@require_admin_callback`

### Подтверждения

Критические операции (удаление, закрытие опросов) требуют подтверждения.
